# resumeapp
Resume app generator

## Setup

Make sure to install the dependencies:

```bash
pnpm i -g vercel@latest
```

```bash
# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server on http://localhost:3000

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```
